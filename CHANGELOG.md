# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-19

### Added
- Initial release of SecCheckmate
- 5 comprehensive security checklists:
  - Web Application Security (35 tests)
  - AWS Cloud Security (40 tests)
  - WiFi Network Security (38 tests)
  - Firmware Security (44 tests)
  - LLM/AI Security (44 tests)
- Interactive CLI interface
- Automatic report generation in Markdown format
- Statistical analysis and severity breakdown
- Evidence tracking for each test
- Configuration management via YAML
- Professional documentation and examples
- MIT License

### Features
- ✅ Offline-first design
- ✅ No external dependencies (only PyYAML)
- ✅ Easy customization of checklists
- ✅ Detailed pass rate analysis
- ✅ Clean, professional reports
- ✅ Evidence and notes collection
- ✅ Severity-based test organization
